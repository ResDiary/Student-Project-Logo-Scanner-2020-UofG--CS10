ResDiaryARMenus Documentation
#############################

This is the Documentation for the CS10 Team Project for ResDiary.

ResDiary require a mobile application that can recognise a restaurant logo and in near real-time, display pertinent information about the restaurant to the user.
The goal of the application is to increase the dicoverability of ResDiary's restaurants.
